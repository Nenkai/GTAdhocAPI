◆ start()

Int pdistd::MSailTranscoder::start ( )

Starts transcoding - cellSailRecorderStart & cellSailPlayerStart on a new thread.

This happens on a seperate thread, use isConverting() or progress to check on it

var transcoder = pdistd::MSailTranscoder();
var res = transcoder.open("in.pam", "out.mp4");
if (res != 0)
{
// Failed to open
return;
}
res = transcoder.start();
if (res != 0)
{
// Failed to transcode
transcoder.close();
return;
}
while (obj.isConverting())
Thread::Sleep(1);
transcoder.close();
